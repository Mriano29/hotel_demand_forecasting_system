# 🏨 Smart Hotel Revenue Management System

An intelligent decision-support system for hotels based on Machine Learning, designed to predict occupancy, cancellations, and optimize room pricing.

---

## 📌 Project Overview

This project develops an accessible Revenue Management system for medium and independent hotels, which usually lack advanced tools to optimize revenue.

Using historical booking data, the system provides:

- 📉 Cancellation prediction  
- 🏨 Occupancy forecasting  
- 💰 Optimal room pricing estimation  
- 📊 Actionable insights through a web application  

---

## 🚨 Problem Statement

Small and medium-sized hotels typically manage pricing manually or based on experience, without leveraging their data effectively.

This leads to:

- Lost revenue opportunities  
- Poor occupancy optimization  
- Competitive disadvantage compared to large hotel chains  

This project aims to popularize access to data-driven Revenue Management tools.

- 🔮 Build predictive models for:

## 🧠 Methodology

The project follows an iterative Machine Learning pipeline:

### 1. 📦 Data Cleaning
- Removal of duplicates  
- Handling missing values  
- Outlier detection (ADR)  
- Data type correction  
- Feature engineering (e.g., total_guests)  (To be added)

### 2. 📊 Exploratory Data Analysis (EDA)
- Booking cancellation analysis  
- Hotel type comparison  
- ADR (Average Daily Rate) analysis  
- Correlation matrix  
- Feature redundancy detection (dendrogram)  

### 3. 🧪 Modeling (To be added)
- Classification models (cancellations)  
- Regression models (pricing)  
- Time series models (occupancy forecasting)  
- Model comparison (Scikit-learn, XGBoost, Prophet)  

### 4. 🚀 Deployment (To be added)
- REST API with FastAPI  
- Frontend with Next.js  
- Real-time prediction serving
